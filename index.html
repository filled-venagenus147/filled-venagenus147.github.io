🔍 What is Foliacode?

Foliacode is a tool for Minecraft server administrators. It checks your existing Bukkit plugins to see if they work with the Folia server software. Folia changes how servers handle tasks, which can cause older plugins to break. This tool finds those issues before you run them on your live server.

🛠 System Requirements

Before you run Foliacode, ensure your computer meets these requirements:

  • Operating System: Windows 10 or Windows 11.
  • Java: You must have Java 21 installed. The tool relies on this specific version to function correctly.
  • Memory: At least 4GB of RAM is recommended for scanning large plugin folders.

📥 How to Download and Install

Follow these steps to obtain the tool:

  1. Visit the official download page.
  2. Look for the latest release version at the top of the list.
  3. Find the files section under the release notes.
  4. Click the file ending in .exe to start the download.
  5. Save the file to your desktop for easy access.

⚙️ Running the Analyzer

Foliacode runs through your command prompt. Do not worry if you have never used this before. Follow these instructions:

  1. Open your Windows Start menu and type "cmd" then press Enter.
  2. Navigate to the folder where you saved the Foliacode file. Type "cd Desktop" and press Enter.
  3. Type the name of the program followed by the word "analyze" and the name of your plugin file. For example: foliacode.exe analyze my-plugin.jar.
  4. Press Enter to start the scan.
  5. Read the output on your screen. The tool lists any errors, warnings, or potential crashes that the plugin might cause on a Folia server.

💡 Understanding the Results

The tool provides a report after it finishes the scan:

  • CRITICAL: This means the plugin will likely crash your server. You should not use these plugins on Folia.
  • HIGH: This indicates a safety issue. The plugin might run, but it could corrupt your world data or cause unexpected behavior.
  • MEDIUM: These are minor points to investigate. They might not stop the plugin, but they can affect performance.

📝 Frequently Asked Questions

Do I need to be a programmer? No. The tool provides human-readable reports that tell you exactly why a plugin fails.

Can I repair the plugins myself? Foliacode gives you the specific location of the code that causes the issue. If you have access to the source code, you can use these hints to modify the plugin.

Is it safe to run? Yes. Foliacode only reads your plugin files. It does not modify or delete anything on your computer.
